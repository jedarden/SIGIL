### 1. Install Manifest Creation
**Status:** ✅ VERIFIED

The install manifest is correctly created at `~/.sigil/install-manifest.toml` during `sigil init`.

**Manifest Structure:**
```toml
[binary]
symlinks = []

[hooks]
git_credential = false
ssh_config = false
docker_config = false

[canaries]
monitoring_active = false

[runtime]

[vault]
path = "/home/user/.sigil/vault"
```

**Implementation Location:**
- Module: `crates/sigil-core/src/install_manifest.rs`
- Usage: Automatically created during vault initialization

### 2. Dry-Run Mode
**Status:** ✅ VERIFIED

The `--dry-run` flag correctly previews what would be removed without making changes.

**Test Results:**
- Shows "Would remove:" messages for each item
- Preserves all files and directories
- Works with all other uninstall flags

**Example Output:**

### 3. Hooks-Only Mode
**Status:** ✅ VERIFIED

The `--hooks-only` flag removes only hook configurations while preserving vault and runtime artifacts.

**Behavior:**
- Removes Claude Code hooks from settings.json
- Removes git credential helper configuration
- Removes SSH config entries
- Preserves vault directory
- Preserves runtime artifacts

**Test Results:**
- Vault directory remains intact
- No impact on vault data
- Hooks properly removed from configuration files

### 4. Keep-Vault Mode
**Status:** ✅ VERIFIED

The `--keep-vault` flag removes everything except the vault directory (this is also the default behavior).

**Behavior:**
- Removes identity.age file
- Removes install-manifest.toml
- Removes all non-vault directories
- Preserves vault directory with all secrets

**Test Results:**
```
Before: 4 items (identity.age, install-manifest.toml, vault/, etc.)
After: 1 item (vault/)
Status: ✅ Vault preserved as expected
```

### 5. Purge Mode
**Status:** ✅ VERIFIED

The `--purge` flag removes ALL SIGIL data including the vault, after explicit confirmation.

**Behavior:**
- Shows warning: "WARNING: This will remove ALL SIGIL data including your vault!"
- Requires user to type "yes" to confirm
- Aborts if any other input is provided
- Removes entire .sigil directory when confirmed

**Test Results:**
- With "no" input: Aborted, no files removed ✅
- With "yes" input: Complete removal, .sigil directory deleted ✅
